# Configuraciones Servicios Docker

## 0. Comandos inicio Docker:
* Instalar Docker
    ```bash
        sudo dnf install docker docker-compose
    ```
* Arrancar Docker
    ```bash
        sudo systemctl enable --now docker
    ```
* Arrancar Servicios docker-compose Docker
    ```bash
        sudo docker compose up -d
    ```
